Welcome to NetBreach - Hacking Software

This software combines two popular hacking tools, Nmap and Hydra, to provide a powerful solution for network penetration testing.

Installation Instructions

  1. Clone the repository: git clone https://github.com/WilSegale/NetBreach.git
  2. Run the setup script: sudo python3 setup.py

This file provides detailed information on how to use the program to its full capacity.

Troubleshooting SSH: Remote Host Identification Changed

If you encounter the following error message when connecting to an SSH server:

        @@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
        @    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
        @@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@

You can delete the hashes connected to previously used SSH hosts by editing the known_hosts file.
